# Keyboard Shortcuts

Snow provides keyboard shortcuts for efficient navigation and paper management. All shortcuts are case-sensitive. Shortcuts don't work when typing in input fields, textareas, or content-editable elements.

## Navigation

| Shortcut | Action |
|----------|--------|
| Shift+H | Previous tab |
| Shift+L | Next tab |
| Shift+K | Previous set |
| Shift+J | Next set |
| j | Next paper |
| k | Previous paper |

## Paper Decisions

| Shortcut | Action |
|----------|--------|
| a | Open accept criterion dialog |
| r | Open reject criterion dialog |
| u | Clear decision for selected paper |

## Bidding

| Shortcut | Action |
|----------|--------|
| b + b | Bid selected paper |
| b + u | Unbid selected paper |
| b + a | Rerun bidding algorithm |

## Snowballing

| Shortcut | Action |
|----------|--------|
| s + s | Snowball (backward & forward) selected paper |
| s + a | Snowball (backward & forward) all consensus-accepted papers |
| s + f + s | Snowball forward selected paper |
| s + f + a | Snowball forward all remaining papers |
| s + b + s | Snowball backward selected paper |
| s + b + a | Snowball backward all remaining papers |

## Filters & View

| Shortcut | Action |
|----------|--------|
| f + a | Toggle accepted chip (show/hide accepted papers) |
| f + u | Toggle undecided chip (show/hide undecided papers) |
| f + r | Toggle rejected chip (show/hide rejected papers) |
| f + f | Toggle perspective (Researcher ↔ Majority) |
| f + s | Cycle sort field (author → title → venue → criterion) |
| v + v | Toggle selected paper details |
